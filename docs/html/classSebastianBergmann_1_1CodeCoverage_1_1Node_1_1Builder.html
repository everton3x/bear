Builds an array representation of the directory structure.

For instance,

Array ( [Money.php] => Array ( ... )

[MoneyBag.php] => Array ( ... ) )

is transformed into

Array ( [.] => Array ( [Money.php] => Array ( ... )

[MoneyBag.php] => Array ( ... ) ) )
